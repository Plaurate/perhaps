1.	Līmenis
Viss ļoti vienkārši, galvenais iziet caur durvīm uz nākamo.
2.	Līmenis
Tas pats bet jāatrod atslēgas
3.	Līmenis
Pa istabu jāatrod kods lai attaisītu durvis
4.	Līmenis
Durvis ir aizbarikadētas, vajag atrast āmuru lai tās atvērtu
5.	Līmenis
Durvis neeksistē, ar to pašu āmuru jāizlauž caurums sienā.
6.	Līmenis
Izslēdzas gaisma, mūzika pazūd. Nākamie daži līmeņi ir tukši bez nekā
7.	Līmenis
Uz sienas parādās dīvains raksts, katrā nākamajā telpā viņš papildinas
Pēdējais līmenis
